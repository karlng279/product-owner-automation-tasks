# Archive

This directory stores archived documents and old versions of tracking files.

## Purpose

- Archive old NOW.md entries (monthly)
- Archive proposal documents after implementation
- Keep historical context without cluttering main directory

## Naming Convention

- `now-YYYY-MM.md` - Archived NOW.md from a specific month
- `proposal-*.md` - Original proposal documents

## Maintenance

Archive monthly:
1. Copy current NOW.md to `archive/now-YYYY-MM.md`
2. Start fresh NOW.md for new month
3. Keep last 6-12 months of archives
4. Delete older archives or compress
