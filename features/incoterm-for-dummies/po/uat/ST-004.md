## TC-001: View responsibility diagram with all stages (P0)

**Scenario:** User sees complete responsibility diagram on detail page
**Precondition:** User is on an Incoterm detail page

| Step | Description |
|------|-------------|
| **Given** | I am viewing the FOB detail page |
| **When** | I scroll to the responsibility diagram section |
| **Then** | I see a horizontal timeline showing all shipping stages from Seller's Premises to Buyer's Premises with the transfer point clearly marked |

- **AC References:** AC-001, AC-002, AC-003, AC-005
- **Test Result:** Not Tested
- **Notes:** Verify all 7 stages are shown.
