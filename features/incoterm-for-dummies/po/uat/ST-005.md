## TC-001: View two-column responsibility breakdown (P0)

**Scenario:** User sees seller and buyer responsibilities side by side
**Precondition:** User is on an Incoterm detail page

| Step | Description |
|------|-------------|
| **Given** | I am viewing the CIF detail page |
| **When** | I scroll to the responsibilities section |
| **Then** | I see a two-column layout showing "Seller's Responsibilities" and "Buyer's Responsibilities" with categorized items for Costs, Risks, and Tasks |

- **AC References:** AC-001, AC-002, AC-003, AC-004
- **Test Result:** Not Tested
- **Notes:** Core functionality for understanding responsibilities.

## TC-002: Expand/collapse responsibility categories (P1)

**Scenario:** User can expand and collapse sections
**Precondition:** User is viewing the responsibilities section

| Step | Description |
|------|-------------|
| **Given** | I am viewing the responsibilities section with collapsed categories |
| **When** | I click on the "Costs" category header |
| **Then** | The Costs section expands to show all cost-related responsibilities |

- **AC References:** AC-006
- **Test Result:** Not Tested
- **Notes:** Accordion behavior test.

## TC-003: Mobile default state is collapsed (P1)

**Scenario:** Sections are collapsed by default on mobile
**Precondition:** User is on mobile device

| Step | Description |
|------|-------------|
| **Given** | I am viewing the responsibilities section on a mobile device |
| **When** | The section loads |
| **Then** | All category sections (Costs, Risks, Tasks) are collapsed by default |

- **AC References:** AC-007
- **Test Result:** Not Tested
- **Notes:** Different default state for mobile vs desktop.

## TC-004: Keyboard navigation through sections (P2)

**Scenario:** User can navigate with keyboard
**Precondition:** User is using keyboard navigation

| Step | Description |
|------|-------------|
| **Given** | I am focused on the responsibilities section |
| **When** | I press Tab to navigate and Enter/Space to expand |
| **Then** | I can navigate through and expand/collapse all sections using keyboard only |

- **AC References:** AC-006, NFR-002
- **Test Result:** Not Tested
- **Notes:** Accessibility test.
