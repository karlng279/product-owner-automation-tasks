## TC-001: View quiz landing page (P0)

**Scenario:** User views quiz start page with all elements
**Precondition:** User navigates to quiz section

| Step | Description |
|------|-------------|
| **Given** | I navigate to the Quiz section from the main navigation |
| **When** | The quiz landing page loads |
| **Then** | I see a welcoming header, quiz description, difficulty selector, estimated time, and Start Quiz button |

- **AC References:** AC-001, AC-002, AC-003, AC-004, AC-005
- **Test Result:** Not Tested
- **Notes:** Base happy path for quiz start.

## TC-002: Start quiz with default difficulty (P0)

**Scenario:** User starts quiz without selecting difficulty
**Precondition:** User is on quiz landing page

| Step | Description |
|------|-------------|
| **Given** | I am on the quiz landing page without selecting a difficulty |
| **When** | I click the "Start Quiz" button |
| **Then** | I am navigated to the first question with Beginner difficulty (10 questions) |

- **AC References:** AC-006, AC-008, AC-010
- **Test Result:** Not Tested
- **Notes:** Default difficulty is Beginner.
