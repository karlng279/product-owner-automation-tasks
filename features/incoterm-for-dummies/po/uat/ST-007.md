## TC-001: Answer a quiz question correctly (P0)

**Scenario:** User selects correct answer and proceeds
**Precondition:** User has started a quiz

| Step | Description |
|------|-------------|
| **Given** | I am on a quiz question |
| **When** | I select the correct answer and click "Next Question" |
| **Then** | I see immediate feedback showing my answer is correct (green highlight) and proceed to the next question |

- **AC References:** AC-004, AC-007, AC-009, AC-010
- **Test Result:** Not Tested
- **Notes:** Happy path for correct answer.

## TC-002: Answer a quiz question incorrectly (P0)

**Scenario:** User selects incorrect answer
**Precondition:** User has started a quiz

- **Notes:** Feedback for wrong answers.

## TC-003: Progress bar updates (P0)

**Scenario:** Progress indicator reflects quiz progress
