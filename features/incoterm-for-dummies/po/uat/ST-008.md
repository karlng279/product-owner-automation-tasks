## TC-001: View quiz results after completion (P0)

**Scenario:** User completes quiz and sees results
**Precondition:** User has answered all quiz questions

| Step | Description |
|------|-------------|
| **Given** | I have answered the last question in the quiz |
| **When** | The results page loads |
| **Then** | I see my overall score (e.g., "8 out of 10 - 80%"), a performance message, and a visual score indicator |

- **AC References:** AC-001, AC-002, AC-003
- **Test Result:** Not Tested
- **Notes:** Core results display.

---

## TC-002: Review incorrect answers with explanations (P0)

**Scenario:** User reviews answers they got wrong
**Precondition:** User has completed quiz with at least one wrong answer

| Step | Description |
|------|-------------|
| **Given** | I am viewing my quiz results with 2 incorrect answers |
| **When** | I expand the review section |
| **Then** | I see my incorrect answers first, with the question, my answer (in red), correct answer (in green), and a "Learn more" link |

- **AC References:** AC-004, AC-007, AC-008
- **Test Result:** Not Tested
- **Notes:** Educational feedback is critical.

---

## TC-003: Retake quiz with new randomized questions (P1)

**Scenario:** User retakes quiz
**Precondition:** User is viewing quiz results

| Step | Description |
|------|-------------|
| **Given** | I am viewing my quiz results |
| **When** | I click "Retake Quiz" |
| **Then** | I am taken to a new quiz with questions in a different random order |

- **AC References:** AC-005, AC-012
- **Test Result:** Not Tested
- **Notes:** Verify randomization differs from previous attempt.

## TC-004: Learn more link navigates to Incoterm detail (P1)
