## TC-001: Select two Incoterms for comparison (P0)

**Scenario:** User selects two Incoterms to compare
**Precondition:** User is on comparison page

| Step | Description |
|------|-------------|
| **Given** | I am on the Compare page with all 11 Incoterms shown as selectable chips |
| **When** | I click on "FOB" and then "CIF" chips |
| **Then** | Both chips are highlighted as selected and the counter shows "2 of 3 selected" |

- **AC References:** AC-002, AC-003, AC-004, AC-005, AC-008
- **Test Result:** Not Tested
- **Notes:** Base selection flow.

## TC-002: Cannot select more than 3 Incoterms (P0)
