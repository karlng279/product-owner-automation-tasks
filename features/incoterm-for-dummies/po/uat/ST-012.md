## TC-001: View comparison table with all attributes (P0)

**Scenario:** User sees complete comparison table
**Precondition:** User has selected FOB and CIF and clicked Compare

| Step | Description |
|------|-------------|
| **Given** | I have selected FOB and CIF and clicked "Compare Now" |
| **When** | The comparison view loads |
| **Then** | I see a table with FOB and CIF as columns, comparing Full Name, Transport Mode, Risk Transfer, Cost Transfer, Seller Responsibilities, Buyer Responsibilities, Insurance, and Best Used When |

- **AC References:** AC-001, AC-002, AC-005
- **Test Result:** Not Tested
- **Notes:** Core comparison display.

## TC-003: Remove Incoterm from comparison (P1)

**Scenario:** User removes one Incoterm from view
**Precondition:** User is comparing 3 Incoterms

| Step | Description |
|------|-------------|
| **Given** | I am comparing FOB, CIF, and EXW |
| **When** | I click the "X" on the CIF column header |
| **Then** | CIF is removed and I see only FOB and EXW in the comparison |

- **AC References:** AC-003, AC-007
- **Test Result:** Not Tested
- **Notes:** Column removal.
