# ST-002: Acceptance Criteria

**Story:** Browse all Incoterms overview
**USL Reference:** [ST-002 in usl.md](../usl.md#st-002-browse-all-incoterms-overview)
**Last Updated:** 2026-01-23

---

## UI Elements

- **AC-001:** Overview page displays all 11 Incoterms 2020 in a grid or list layout.
- **AC-002:** Each Incoterm card shows:
  - Incoterm code (e.g., FOB, CIF, EXW)
  - Full name (e.g., "Free On Board")
  - One-line summary description
  - Transport mode indicator (Sea/Multimodal icon)
- **AC-003:** Incoterms are visually grouped by transport mode:
  - Sea and Inland Waterway Transport (FAS, FOB, CFR, CIF)
  - Any Mode of Transport (EXW, FCA, CPT, CIP, DAP, DPU, DDP)
- **AC-004:** Page header shows "All Incoterms 2020" as the title.

---

## UI Behavior

- **AC-005:** Clicking on an Incoterm card navigates to that Incoterm's detail page.
- **AC-006:** On hover, Incoterm cards show a subtle highlight effect.
- **AC-007:** Cards animate in with a staggered fade effect on page load.

---

## Logic

- **AC-008:** All 11 Incoterms 2020 are displayed: EXW, FCA, FAS, FOB, CFR, CIF, CPT, CIP, DAP, DPU, DDP.
- **AC-009:** Incoterms data is loaded from static JSON (no API call required).

---

## Special Notes

- **AC-010:** Transport mode groups have descriptive subheadings explaining the grouping.
- **AC-011:** Page is responsive: 4 cards per row on desktop, 2 on tablet, 1 on mobile.
- **AC-012:** Include a brief intro paragraph explaining what Incoterms are.

---

## Non-Functional Requirements

- **NFR-001:** Page renders within 500ms.
- **NFR-002:** All card images/icons are optimized and under 50KB total.
