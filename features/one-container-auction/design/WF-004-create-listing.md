**Page Structure:**
- **Header:** Standard.
- **Main Content:** Centered form container (max-width 800px).
- **Footer:** Standard.

**Content Zones:**

1. **Zone A (Form Header):**
   - **Elements:**
     - [Title] "Sell a Container"
     - [Breadcrumbs] "Dashboard > Create Listing"
   - **Purpose:** Context.

2. **Zone B (Container Details Section):**
   - **Elements:**
     - [Label] "Container Type" -> [Select Dropdown] (20ft, 40ft...)
     - [Label] "Condition" -> [Radio Group] (New, Used, Damaged)
     - [Label] "Location" -> [Input Text]
     - [Label] "Description" -> [Textarea]
   - **Purpose:** Define the asset.

3. **Zone C (Media Section):**
   - **Elements:**
     - [Label] "Photos"
     - [Input] "Image URL" (MVP) OR [Upload Area] (Phase 2)
     - [Preview] (Shows image after URL entered)
   - **Purpose:** Visual proof.

4. **Zone D (Auction Settings):**
   - **Elements:**
     - [Label] "Start Price (â‚¬)" -> [Number Input]
     - [Label] "Buy Now Price (Optional)" -> [Number Input]
     - [Label] "Auction End Time" -> [Date/Time Picker]
   - **Purpose:** Set sales parameters.

5. **Zone E (Actions):**
   - **Elements:**
     - [Button] "Cancel" (Secondary)
     - [Button] "Create Listing" (Primary)
   - **Purpose:** Submission.

### Tablet & Mobile

**Layout Changes:**
- **Form:** Full width.
- **Inputs:** Stacked vertically.

---

## Interaction Notes

- **Validation:** Fields turn red with error text if invalid on blur.
- **Submission:** "Create Listing" button shows spinner state.
- **Success:** Redirects to Dashboard with toast "Listing published successfully".
